## slice (usage rate more than array)

slice = arr[x:x]  
slice is a view of a array

```go
package main
import "fmt"
func s() {
	arr := [...]int{0,1,2,3,4,5,6,7}
	s1 := arr[2:6]
	s2 := s1[3:5]
	fmt.Println(s1) // [2,3,4,5]
	fmt.Println(s2) // [5,6]
}
```

![slice](slice.jpg)
the len of s1 is 4, but the capacity is 6, so we can get s2
